# Speech to text

## Parameters


- **model**
  - Type: string
  - Required: false
  - Options: whisper, fast-whisper, gladia
  - Default: fast-whisper

- **audio**
  - Type: string
  - Required: true

- **diarization**
  - Type: boolean
  - Required: false

- **type**
  - Type: string
  - Required: false
  - Options: tiny, base, small, medium, large-v1, large-v2
  - Default: base
