# Text to speech
POST *https://api.turing.sh/audio/tts*
## Parameters


- **model**
  - Type: string
  - Required: true
  - Options: google, elevenlabs
  - Default: google

- **voice**
  - Type: string
  - Required: true
  - Description: Voice to use (elevenlabs only)
  - Default: adam

- **text**
  - Type: string
  - Required: true
  - Description: Text to convert to speech

- **language**
  - Type: string
  - Required: true
  - Description: Language code to use (google only)
  - Default: en

- **slow**
  - Type: boolean
  - Required: false
  - Description: Speak slowly (google only)

- **stream**
  - Type: boolean
  - Required: false
