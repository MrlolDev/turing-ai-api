# Text to speech

## Parameters


- **model**
  - Type: string
  - Required: true
  - Options: google, elevenlabs
  - Default: google

- **voice**
  - Type: string
  - Required: true

- **text**
  - Type: string
  - Required: true

- **language**
  - Type: string
  - Required: true
  - Default: en

- **slow**
  - Type: boolean
  - Required: false

## Examples

<CodeGroup title="Text to speech example">
  ```typescript
  import axios from "axios";
  (async () => {
    let response = await axios({
      method: "post",
      url: 'https://api.turing.sh/audio/tts',
      headers: {
        "Content-Type": "application/json",
        Authorization: "Bearer YOUR_API_KEY",
      },
      data: {
  "model": "google",
  "voice": "string",
  "text": "string",
  "language": "en"
},
    })
  })();
  ```
