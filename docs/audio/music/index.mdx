# Music generation
POST *https://api.turing.sh/audio/music*

## Parameters


- **prompt**
  - Type: string
  - Required: true

- **model**
  - Type: string
  - Required: false
  - Options: small, medium, melody, large
  - Default: small

- **duration**
  - Type: number
  - Required: false
  - Default: 8

- **stream**
  - Type: boolean
  - Required: false
