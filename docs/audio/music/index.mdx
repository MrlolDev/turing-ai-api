# Music generation

## Parameters


- **prompt**
  - Type: string
  - Required: true

- **model**
  - Type: string
  - Required: false
  - Options: small, medium, melody, large
  - Default: small

- **duration**
  - Type: number
  - Required: false
  - Default: 8
