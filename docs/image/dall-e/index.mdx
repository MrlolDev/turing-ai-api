# Dall-e 2
POST *https://api.turing.sh/image/dall-e*
## Parameters


- **prompt**
  - Type: string
  - Required: false

- **number**
  - Type: number
  - Required: true
  - Options: 1, 2, 3, 4

- **size**
  - Type: string
  - Required: false
  - Options: 512x512, 256x256, 1024x1024
  - Default: 512x512

- **image**
  - Type: string
  - Required: false
  - Description: Image you want to vary
