# Controlnet

## Parameters


- **prompt**
  - Type: string
  - Required: false

- **model**
  - Type: string
  - Required: true
  - Options: normal, canny, hough, hed, depth2img, pose, seg
  - Default: normal

- **image**
  - Type: string
  - Required: false
