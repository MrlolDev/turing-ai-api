# Controlnet
POST *https://api.turing.sh/image/controlnet*
## Parameters


- **prompt**
  - Type: string
  - Required: true

- **model**
  - Type: string
  - Required: true
  - Options: normal, canny, hough, hed, depth2img, pose, seg
  - Default: normal

- **image**
  - Type: string
  - Required: true

- **stream**
  - Type: boolean
  - Required: false
