# Image vision
POST *https://api.turing.sh/image/vision*
## Parameters


- **model**
  - Type: array
  - Required: true
  - Description: Model to use for the analysis
  - Options: blip2, ocr
  - Default: blip2

- **image**
  - Type: string
  - Required: true
  - Description: Image URL for the model to process
