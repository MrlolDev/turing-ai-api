# MJ

## Parameters


- **prompt**
  - Type: string
  - Required: false

- **model**
  - Type: string
  - Required: false
  - Options: 1, 2, 3, 4, 5, 5.1, 5.2, niji
  - Default: 5.2

- **action**
  - Type: string
  - Required: false
  - Options: imagine, variation, upscale
  - Default: imagine

- **number**
  - Type: number
  - Required: false
  - Options: 0, 1, 2, 3

- **id**
  - Type: string
  - Required: false

- **stream**
  - Type: boolean
  - Required: false
  - Default: true
