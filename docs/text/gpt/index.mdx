# OpenAI Models

## Parameters


- **messages**
  - Type: array
  - Required: true

- **model**
  - Type: string
  - Required: true
  - Options: gpt-3.5-turbo, gpt-4
  - Default: gpt-3.5-turbo

- **max_tokens**
  - Type: number
  - Required: false
  - Default: 512

- **temperature**
  - Type: number
  - Required: false
  - Default: 0.9

- **plugins**
  - Type: array
  - Required: false
  - Default: 

- **stream**
  - Type: boolean
  - Required: false
  - Default: true
