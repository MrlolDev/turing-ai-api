# Anthropic Models

## Parameters


- **messages**
  - Type: array
  - Required: true

- **model**
  - Type: string
  - Required: false
  - Options: claude-1, claude-instant-1
  - Default: claude-instant-1

- **max_tokens**
  - Type: number
  - Required: false
  - Default: 512

- **temperature**
  - Type: number
  - Required: false
  - Default: 0.9

- **stream**
  - Type: boolean
  - Required: false
