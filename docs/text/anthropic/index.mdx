# Anthropic Models

## Parameters


- **messages**
  - Type: array
  - Required: false

- **model**
  - Type: string
  - Required: true
  - Options: claude-v1, claude-instant
  - Default: claude-instant

- **max_tokens**
  - Type: number
  - Required: false
  - Default: 512

- **temperature**
  - Type: number
  - Required: false
  - Default: 0.9
