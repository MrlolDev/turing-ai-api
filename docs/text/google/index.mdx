# Google Models

## Parameters


- **messages**
  - Type: array
  - Required: true

- **model**
  - Type: string
  - Required: true
  - Options: chat-bison
  - Default: chat-bison

- **max_tokens**
  - Type: number
  - Required: false
  - Default: 512

- **temperature**
  - Type: number
  - Required: false
  - Default: 0.9
