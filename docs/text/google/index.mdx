# Google Models
POST *https://api.turing.sh/text/google*

## Parameters


- **messages**
  - Type: array
  - Required: true

- **model**
  - Type: string
  - Required: false
  - Options: chat-bison
  - Default: chat-bison

- **max_tokens**
  - Type: number
  - Required: false
  - Default: 512

- **temperature**
  - Type: number
  - Required: false
  - Default: 0.9

- **id**
  - Type: string
  - Required: false
  - Description: ID of the conversation (used for data saving)
  - Default: 38a92213-d283-4a43-becd-f489e543f085

## Response


- **cost**
  - Type: number
  - Description: Cost of the request in USD

- **result**
  - Type: string
  - Description: Result of the request

- **done**
  - Type: boolean
  - Description: Whether the request is done or not

- **id**
  - Type: string
  - Description: ID of the conversation (used for data saving)
