# Huggingface based models

## Parameters


- **messages**
  - Type: array
  - Required: false

- **prompt**
  - Type: string
  - Required: false

- **chat**
  - Type: boolean
  - Required: false

- **model**
  - Type: string
  - Required: true
  - Options: OpenAssistant/oasst-sft-4-pythia-12b-epoch-3.5

- **stop**
  - Type: string
  - Required: false
