# GroqCloud models
POST *https://api.turing.sh/text/groq*

## Parameters


- **messages**
  - Type: array
  - Required: true

- **model**
  - Type: string
  - Required: false
  - Options: mixtral-8x7b-32768
  - Default: mixtral-8x7b-32768

- **max_tokens**
  - Type: number
  - Required: false
  - Default: 512

- **temperature**
  - Type: number
  - Required: false
  - Default: 0.9

- **id**
  - Type: string
  - Required: false
  - Description: ID of the conversation (used for data saving)
  - Default: 7ab7480c-8e5d-4177-89a1-f90444798ae5

## Response


- **result**
  - Type: string

- **done**
  - Type: boolean

- **cost**
  - Type: number

- **finishReason**
  - Type: string

- **id**
  - Type: string
  - Description: ID of the conversation (used for data saving)
