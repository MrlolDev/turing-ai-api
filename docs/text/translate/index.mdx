# Translate text

## Parameters


- **text**
  - Type: string
  - Required: true

- **from**
  - Type: string
  - Required: false
  - Default: auto

- **to**
  - Type: string
  - Required: true

- **ai**
  - Type: string
  - Required: true
  - Options: google, microsoft
  - Default: google
