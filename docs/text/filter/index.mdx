# Text Filter
POST *https://api.turing.sh/text/filter*
## Parameters


- **text**
  - Type: string
  - Required: true
  - Description: Text you want to filter

- **filters**
  - Type: array
  - Required: true
  - Description: Filters you want to apply
  - Options: nsfw, cp, toxicity

- **stream**
  - Type: boolean
  - Required: false

## Examples
