# OpenChat models
POST *https://api.turing.sh/text/openchat*

## Parameters


- **messages**
  - Type: array
  - Required: true

- **model**
  - Type: string
  - Required: false
  - Options: openchat_v3.2
  - Default: openchat_v3.2

- **max_tokens**
  - Type: number
  - Required: false
  - Default: 512

- **temperature**
  - Type: number
  - Required: false
  - Default: 0.9
