# OpenChat models
POST *https://api.turing.sh/text/openchat*

## Parameters


- **messages**
  - Type: array
  - Required: true

- **model**
  - Type: string
  - Required: false
  - Options: openchat_v3.2_mistral
  - Default: openchat_v3.2_mistral

- **max_tokens**
  - Type: number
  - Required: false
  - Default: 512

- **temperature**
  - Type: number
  - Required: false
  - Default: 0.9

- **id**
  - Type: string
  - Required: false
  - Description: ID of the conversation (used for data saving)
  - Default: 55922e55-e0bf-44c8-8688-ac7173d55694

- **autoSystemMessage**
  - Type: boolean
  - Required: false
  - Description: Send system messages automatically
  - Default: true

## Response


- **result**
  - Type: string

- **done**
  - Type: boolean

- **cost**
  - Type: number

- **finishReason**
  - Type: string

- **id**
  - Type: string
  - Description: ID of the conversation (used for data saving)
