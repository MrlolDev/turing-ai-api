# Pawan.krd models
POST *https://api.turing.sh/text/pawan*

## Parameters


- **messages**
  - Type: array
  - Required: true

- **model**
  - Type: string
  - Required: false
  - Options: zephyr-7b-beta, pai-001-light-beta
  - Default: zephyr-7b-beta

- **max_tokens**
  - Type: number
  - Required: false
  - Default: 512

- **temperature**
  - Type: number
  - Required: false
  - Default: 0.9

- **id**
  - Type: string
  - Required: false
  - Description: ID of the conversation (used for data saving)
  - Default: 9bed6ed8-6040-4cd1-84a3-68a414d7a731

- **autoSystemMessage**
  - Type: boolean
  - Required: false
  - Description: Send system messages automatically
  - Default: true

## Response


- **result**
  - Type: string

- **done**
  - Type: boolean

- **cost**
  - Type: number

- **finishReason**
  - Type: string

- **id**
  - Type: string
  - Description: ID of the conversation (used for data saving)
